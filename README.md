# rn-aws-s3

React Native AWS S3 client (some utils)

## Installation

```sh
npm install rn-aws-s3
```

## Usage


```js
import { putObject } from 'rn-aws-s3';
await putObject({ filePath, bucketName, fileKey, region, accessKey, secretKey })
```
